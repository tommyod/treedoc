# treedoc

Minimalistic Python documentation in a tree structure.

## Examples

Install using `pip install treedoc`.

### Terminal

```bash
$ treedoc collections.Counter
$ treedoc itertools
$ treedoc numpy.linalg 
$ treedoc --help
```

### Terminal

```python
>>> from treedoc import treedoc
>>> treedoc(list)
>>> import functools
>>> treedoc(functools)
```
